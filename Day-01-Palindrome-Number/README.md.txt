# Day 1 - Palindrome Number

## 🧩 Problem
Given an integer x, return true if x is a palindrome, and false otherwise.

## 💡 Approach
- Reversed the number using mathematical operations (modulus and division)
- Compared the reversed number with the original
- Handled edge case: negative numbers (e.g., -121 is not a palindrome)

## ⚙️ Complexity
- Time Complexity: O(log n)
- Space Complexity: O(1)

## 🚀 Example
Input: x = 121  
Output: true  

Input: x = -121  
Output: false  

## 📌 Key Learning
- How to reverse a number without converting to string
- Importance of handling edge cases
